# Number Guessing Game

This Python game generates a random number and asks the user to guess it.

## Features
- Random number generation
- User input
- Hint system (higher or lower)

## Technologies Used
- Python
- Random module

## How to Run
1. Run the Python script
2. Enter your guesses
3. The program will guide you until you guess correctly

## Output Example
Guess the number: 5  
Too low! Try again.  
Guess the number: 8  
Correct! You guessed the number.
